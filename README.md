ColorBlind App
===============

Simple Cordova app about colorblindness

Build
-----

```bash
$ cordova prepare
$ grunt build
```

Run & watch
-----------

```bash
$ grunt
```

Add platform or Plugin
----------------------

`./platforms/` & `./plugins` are ignored by git. So don't forget to save any change.

```bash
$ cordova platform add <platform> --save
$ cordova plugin add <plugin> --save
```
